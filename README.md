# raymondkarlsen.no

Min personlige landingsside. Kort bio og inngang til hobbyprosjektene mine.

**Live:** [raymondkarlsen.no](https://raymondkarlsen.no)

## Hva er dette?

En enkel, statisk side bygget fra bunnen av med HTML, CSS og litt JavaScript. Ingen rammeverk, ingen byggeverktøy — bare filer servert rett fra GitHub Pages.

Prosjektet er like mye en læringsreise i Git, GitHub og webutvikling som det er en nettside.

## Struktur

    ├── index.html      # Alt innhold
    ├── css/style.css   # All styling, inkl. animasjoner
    ├── js/main.js      # Foreløpig tom, klar for fremtidig bruk
    ├── assets/         # Bilder og andre ressurser
    └── CNAME           # Kobler GitHub Pages til raymondkarlsen.no

## Design

- **Fonter:** Anton (overskrifter) + Space Grotesk (brødtekst)
- **Farger:** Snow-bakgrunn med blålilla palett (Ultrasonic Blue, Persian Blue, Twilight Indigo, Thistle) — hentet fra en fotballdrakt, naturligvis
- **Bevegelse:** flytende geometriske former, bølgelinje med fargeovergang, roterende sykkelhjul og svaiende drakter på prosjektkortene

## Publisering

Pushes til `main` publiseres automatisk via GitHub Pages.
